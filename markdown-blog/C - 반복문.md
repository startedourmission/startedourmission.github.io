# 반복문 (Loop)

C 언어에는 주로 세 가지 종류의 반복문이 있습니다:

### 1. `for` 문
특정 횟수만큼 반복하거나, 초기화, 조건, 증감식이 명확할 때 사용합니다.

```c
for (초기화; 조건; 증감) {
    // 반복할 코드
}
```

### 2. `while` 문
조건이 참(true)인 동안 반복합니다. 조건이 처음부터 거짓(false)이면 한 번도 실행되지 않을 수 있습니다.

```c
while (조건) {
    // 반복할 코드
}
```

### 3. `do-while` 문
코드를 최소 한 번 실행한 후 조건이 참(true)인 동안 반복합니다.

```c
do {
    // 반복할 코드
} while (조건);
```
