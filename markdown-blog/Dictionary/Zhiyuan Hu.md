싱가포르국립대학교(NUS) 박사과정(2023-2026)이자 MIT 연구원(2025-2026). 지도교수는 [[Bryan Hooi]], See-Kiong Ng, Anh Tuan Luu.

대표 연구는 **MAgIC** -- 다중 LLM 에이전트의 인지, 적응성, 합리성, 협업 능력을 조사한 연구입니다. 대화 시스템 개선을 위한 LLM 기반 사용자 시뮬레이터 연구도 진행했습니다. 전반적으로 LLM 에이전트의 사회적·대화적 능력에 관심을 가진 연구자입니다.

[[EvoArena - Tracking Memory Evolution for Robust LLM Agents in Dynamic Environments|EvoArena]] 논문에서 NUS/MIT 쪽 기여자로 참여했으며, 에이전트 벤치마크 설계 전반에 기여했습니다.
