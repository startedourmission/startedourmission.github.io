Zirui Guo는 홍콩대학교(HKU) [[황차오]] 교수 지도 아래 HKUDS Lab에서 박사과정을 밟고 있는 연구자입니다. 그래프 기반 RAG(검색 증강 생성) 아키텍처를 전문으로 합니다.

2024년 발표한 LightRAG(arXiv:2410.05779)의 1저자입니다. LightRAG는 단순 청크 검색을 넘어 지식 그래프를 활용한 듀얼 레벨 검색(저수준 엔티티 + 고수준 개념)을 RAG에 통합한 프레임워크로, 공개 직후 GitHub 스타 36,000개 이상을 받으며 오픈소스 RAG 생태계의 기준점 중 하나가 됐습니다. EMNLP 2025 Findings에 채택됐습니다.

2025년 10월 공개한 RAG-Anything(arXiv:2510.12323)은 LightRAG의 멀티모달 확장판입니다. 텍스트만 다루던 기존 RAG의 한계를 넘어 이미지·표·수식·코드를 모두 노드로 두는 듀얼 그래프 구조를 도입했습니다. Cross-modal knowledge graph(비텍스트 콘텐츠 중심)와 text-based knowledge graph(텍스트 중심, LightRAG 방식)를 각각 구축한 뒤 엔티티 정렬로 병합합니다. DocBench와 MMLongBench 두 멀티모달 문서 QA 벤치마크에서 최신 기준선 대비 전반적인 성능 향상을 보였으며, 특히 100페이지 이상 장문서에서 차이가 두드러졌습니다. 공개 직후 GitHub Trending 1위에 올랐습니다.
