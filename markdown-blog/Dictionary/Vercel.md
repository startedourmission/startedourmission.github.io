Vercel은 2015년 기예르모 라우치(Guillermo Rauch)가 ZEIT라는 이름으로 창업해 2020년 현재 사명으로 변경한 미국 프런트엔드 배포 플랫폼 회사다. 회사가 직접 메인테인하는 React 메타 프레임워크 Next.js를 중심으로, 글로벌 엣지 네트워크 기반 호스팅·서버리스 함수·이미지 최적화 등을 제공한다.

AI 도구 라인업으로는 자연어 UI 생성 도구 [[v0]], 모델 호출 추상화 라이브러리 Vercel AI SDK, 다중 모델 라우팅 게이트웨이 AI Gateway 등을 운영한다. 프런트엔드 개발자가 LLM 통합 기능을 가장 빠르게 만들 수 있는 스택으로 자주 추천된다.

2026년 4월에는 라이브러리 의존성을 통한 공급망 보안 사고를 겪으며 Next.js·Vercel 생태계 전체의 보안 거버넌스 논의를 촉발한 주체이기도 하다.
