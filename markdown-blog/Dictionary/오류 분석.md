# 오류 분석

오류 분석(Error Analysis)은 머신러닝 모델이 틀린 예측을 한 사례들을 체계적으로 살펴보고, 모델의 약점과 개선 방향을 파악하는 진단 프로세스이다.

## 핵심

- 오분류된 샘플을 직접 검토하여 오류 패턴을 식별한다
- 어떤 유형의 오류가 가장 많은지 파악해 우선순위를 정한다
- 혼동 행렬(confusion matrix)로 클래스 간 오류 분포를 시각화한다
- 오류의 원인이 데이터 품질, 특징 부족, 모델 복잡도 중 어디에 있는지 진단한다
- 체계적인 오류 분석은 편향-분산 분석과 함께 모델 개선의 핵심 도구이다
