상하이인공지능실험실(Shanghai Artificial Intelligence Laboratory) 대형 모델 센터 연구 책임자. 칭화대 학사(2015), 홍콩중문대 박사(2019).

인턴(Intern) 시리즈를 이끄는 핵심 인물. 주요 기여:

- **OpenMMLab** (2018~) - GitHub 스타 12만+, 140개국 연구자가 사용하는 컴퓨터비전 알고리즘 오픈소스 생태계
- **MMDetection / MMDetection3D** - 객체 탐지·3D 탐지 표준 프레임워크
- **InternLM** - 상하이AI랩 대표 대형 언어 모델 시리즈
- **InternVL** - 시각-언어 멀티모달 파운데이션 모델
- **Intern-S1/S2** - 수학·과학 추론 특화 모델

논문 70편 이상, 피인용 2만 7천+ (2026 기준). [[Scalable Visual Pretraining for Language Intelligence]] 논문의 교신저자.
