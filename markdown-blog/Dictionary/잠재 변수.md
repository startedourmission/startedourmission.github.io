# 잠재 변수

잠재 변수(Latent Variable)는 데이터에서 직접 관측되지 않고 관측 변수를 통해 간접적으로 추론해야 하는 숨겨진(hidden) 변수이다. 많은 생성 모델에서 데이터 생성 과정의 핵심 요소이다.

## 핵심

- 가우시안 혼합 모델에서 각 포인트의 클러스터 소속이 대표적인 잠재 변수이다
- 잠재 변수의 존재로 인해 직접적인 최대 우도 추정이 어려워 EM 알고리즘을 사용한다
- 인자 분석에서 저차원 잠재 요인이 고차원 관측값을 설명한다
- VAE(변분 오토인코더)는 딥러닝에서 잠재 변수를 학습하는 대표적 모델이다
- 잠재 공간(latent space)에서의 표현 학습이 생성 모델의 핵심이다
