홍콩중문대학교(CUHK) 컴퓨터과학공학과 박사과정 1년차입니다. 지도교수는 [[Pheng-Ann Heng]]. 연구 관심사는 LLM의 포스트트레이닝, 멀티모달 AI 평가, 에이전트 시스템의 교집합입니다.

산업계 인턴 경력은 두 곳입니다. 2024년 8월부터 2025년 4월까지 텐센트 YouTu Lab에서, 이후 알리바바 통이랩(Tongyi Lab)에서 연구 인턴으로 일했습니다. 박사과정 중에도 대형 기업 연구소와 지속적으로 협업하는 스타일입니다.

대표 논문으로는 멀티모달 언어모델의 감정 이해를 평가하는 MME-Emotion(ICLR 2026), 세포 생물학 이해를 측정하는 CellVerse(NeurIPS 2025)가 있습니다. GitHub 핸들 `zfkarl`로 활동하며 Orchestra-o1의 공개 코드 저장소도 이 계정에 있습니다.

[[Orchestra-o1 - Omnimodal Agent Orchestration]]에서는 공동 1저자로 참여했습니다. CUHK-PKU-THU-통지대-LIGHTSPEED 5개 기관 협업 연구의 CUHK 측 핵심 기여자입니다.
