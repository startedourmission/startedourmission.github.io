## 개요

단단 궈(Dandan Guo)는 [[지린대학교]] 소속 머신러닝 연구자입니다. 통계적 머신러닝을 기반으로 확률적 방법·심층 생성 모델·표현학습·메타학습을 연구하며, 시계열 모델링·텍스트 분석·소수샷 생성·레이더 표적 인식 등에 적용해왔습니다. ICLR 2024에 표 형식 데이터 표현학습 연구 PTARL을 발표했습니다.

[[GateMem - Benchmarking Memory Governance in Multi-Principal Shared-Memory Agents]]의 교신저자로 참여했습니다.
