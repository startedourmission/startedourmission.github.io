베이징대학교(PKU) School of EECS의 NLP 교수이며, 교육부 전산언어학 핵심실험실(MOE Key Lab of Computational Linguistics) 소속입니다. 연구 분야는 담화 분석, 문서 요약, 텍스트 생성, 키워드 추출, 기계 독해, 정보 추출로 시작해 최근에는 LLM 정렬과 멀티모달 에이전트로 확장되고 있습니다. Google Scholar 기준 인용 12,500여 회로 중국 NLP 그룹 중에서도 누적 영향력이 큰 시니어 PI입니다.

이 그룹은 [[Feifan Song]]의 PRO(Preference Ranking Optimization, AAAI 2024)처럼 인간 선호 정렬 분야에서 일찍 자리잡은 작업과, [[슝웨이민]] 같은 박사·인턴 라인이 산업 LLM 팀과 함께 진행하는 데이터 합성 파이프라인을 동시에 굴립니다. 이번 [[Video2GUI - Synthesizing Large-Scale Interaction Trajectories for Generalized GUI Agent Pretraining|Video2GUI]]에서는 제1 corresponding author로, 학생 1저자 Weimin Xiong과 [[톈하오]] 측 Xiaomi LLM-Core 팀을 잇는 학계 측 책임자 역할을 맡았습니다.

연락 이메일은 lisujian@pku.edu.cn으로, MOE Key Lab 페이지와 PKU EECS 연구진 명단에서 그대로 확인됩니다.
