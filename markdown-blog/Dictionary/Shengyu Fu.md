Microsoft Research 소속으로, LLM 기반 소프트웨어 엔지니어링 에이전트 연구를 이끌고 있습니다. [[FastContext - Training Efficient Repository Explorer for Coding Agents]] 논문에서 교신 저자를 맡아 Microsoft와 상하이교통대학(SJTU)의 공동 연구를 총괄했습니다.

코딩 에이전트의 레포지토리 탐색 분리라는 연구 방향을 설정하고, SFT 초기화에 Claude Sonnet 4.6 참조 모델 궤적을 활용하는 데이터 파이프라인과 GRPO 기반 RL 정제 방법론의 설계를 주도했습니다.
