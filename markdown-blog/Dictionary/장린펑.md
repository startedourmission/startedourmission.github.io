[[장린펑]](张林峰)은 상하이교통대학교 인공지능학원의 조교수이자 박사 지도교수입니다. 칭화대학교 교차정보과학원(IIIS)에서 박사 학위를 받았고, 지금은 EPIC(Efficient and Precision Intelligent Computing) 랩을 이끌고 있습니다.

연구의 중심은 효율적 AI입니다. 지식 증류, 모델 압축, 토큰 절감처럼 큰 모델을 더 적은 연산으로 돌리는 기법을 다룹니다. CCF-A·CAAI-A급 학회와 저널에 (공동)1저자 또는 교신저자로 40여 편을 냈고 피인용 3000회를 넘겼습니다. Microsoft Fellowship(아시아 12인), World AI Conference Cloud Sail Award 등을 받았고 ACL·NeurIPS·ICLR의 Area Chair를 맡고 있습니다.

[[HLL - Can Agents Cross Humanity's Last Line of Verification]]에서는 [[류둥루이]]와 함께 교신저자로 참여했습니다. 효율적 AI를 연구하던 그가 에이전트 평가 벤치마크에 합류한 것은, 멀티모달 에이전트가 실제 배포 환경에서 얼마나 견고한지를 측정하는 문제가 모델의 실전 가치를 가르는 핵심이라는 판단으로 읽힙니다.
