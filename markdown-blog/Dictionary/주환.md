베이징대학교(Peking University) 소속 AI 연구자. 2026년 단독 저자로 "Think Through a Bottleneck: Hourglass Reasoning for Rigorous Induction"(arXiv:2607.11696)을 발표했다. 귀납적 few-shot 추론에서 컨텍스트 오염이 Self-Refine 성능을 저하시킨다는 문제의식에서 출발해, 추론 단계를 독립 API 호출로 격리하고 압축 심볼릭 상태 $(\phi, T)$만 단계 간에 전달하는 파이프라인을 설계했다.
