Doubao(豆包, 도우바오)는 TikTok 모회사 ByteDance(字节跳动)가 개발하는 LLM 시리즈다. 같은 이름의 챗봇 앱은 2024년 들어 중국 본토 AI 챗봇 사용자 1위로 올라섰으며, 회사 자체 통계로 일간 활성 사용자가 수천만 명 단위에 달한다.

모델 라인은 Doubao 1.5 Pro, Doubao Seed 시리즈로 발전했으며, 통합 추론·비디오·음성 모달을 지원한다. ByteDance가 추천 알고리즘으로 축적한 거대한 사용자 데이터·인프라가 학습 자원으로 그대로 활용되는 점이 강점이다.

가격을 공격적으로 낮추는 전략으로 중국 LLM API 가격 전쟁의 방아쇠를 당긴 회사이기도 하다. 2024년 5월 100만 토큰당 0.8위안(약 0.11달러) 수준의 가격을 발표하면서 [[DeepSeek]] 이전부터 중국 모델 가격 하락을 주도했다.
