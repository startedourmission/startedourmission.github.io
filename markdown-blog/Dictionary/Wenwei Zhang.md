상하이인공지능실험실(Shanghai AI Lab) 영 리서치 사이언티스트. OpenMMLab의 핵심 개발자이자 파운데이션 모델 포스트트레이닝 연구자.

주요 기여:

- **MMDetection / MMDetection3D** - 2020년부터 리드 개발자로 참여. 객체 탐지·3D 탐지 분야 표준 프레임워크
- **Intern-S1/S2** - 수학·과학 추론 파운데이션 모델. Scalable Oversight 기법 적용
- **InternLM2-Chat** - 오픈소스 LLM 평가 리더보드 상위

현재 연구 초점은 파운데이션 모델의 포스트트레이닝과 자기 개선(Self-Improvement/Scalable Oversight). [[Scalable Visual Pretraining for Language Intelligence]] 논문의 공동 제1저자.
