Llama는 Meta가 공개하는 오픈 웨이트 LLM 시리즈다. 2023년 2월 LLaMA 1을 연구용으로 제한 공개한 뒤, Llama 2부터 상업적 사용까지 허용한 라이선스로 풀어 오픈소스 LLM 생태계의 사실상 표준이 되었다.

세대별로 1, 2, 3, 3.1, 3.3, 4를 거치며 파라미터 규모와 컨텍스트를 확장해왔다. [[혼합 전문가 모델|MoE]] 구조를 도입한 Llama 4부터는 수천억 파라미터 급으로 프론티어 모델과 직접 경쟁하는 체급에 올라섰다.

[[Yann LeCun]]이 이끄는 Meta FAIR의 오픈 사이언스 기조가 만든 결과물이며, Vicuna·Alpaca를 비롯한 수많은 파생 모델을 낳았다. 학계와 스타트업이 자체 모델을 만드는 베이스로 가장 많이 선택되는 가족이다.
