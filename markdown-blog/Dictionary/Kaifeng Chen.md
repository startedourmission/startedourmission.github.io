[[Kaifeng Chen]]은 중국 톈진대학교(Tianjin University) 소속 연구자로, 멀티모달 긴 문서 질의응답(multimodal long document QA)과 에이전트 메모리를 연구합니다.

본 논문 [[MARDoc - A Memory-Aware Refinement Agent Framework for Multimodal Long Document QA]]의 제1저자입니다. 긴 문서 QA에서 상호작용 기록을 하나의 거대한 맥락에 계속 쌓는 기존 방식(monolithic context stream)을, 탐색·정제·반성 세 에이전트가 구조화된 메모리로 소통하는 루프로 바꾸는 MARDoc의 설계와 실험을 주도했습니다. 교신저자 [[Qiyao Peng]]과 같은 톈진대 소속입니다.
