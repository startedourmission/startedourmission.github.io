[[Tsachy Weissman]]는 스탠퍼드 대학교 전기공학과 교수입니다. Technion(이스라엘 공대)에서 박사를 받고 HP 연구소를 거쳐 2003년 스탠퍼드 조교수로 합류, 2015년 정교수가 되었습니다. 정보이론 기여로 2013년 IEEE 펠로우에 선정되었습니다.

연구 분야는 정보이론, 데이터 압축, 통계적 신호처리, 머신러닝과 이들의 접점입니다. HP 시절 이산 보편 디노이저(DUDE) 알고리즘의 공동 발명자였고, 최근에는 유전체 데이터 압축·처리에 관심을 둡니다. 스탠퍼드 압축 포럼(Stanford Compression Forum)의 창립 디렉터이기도 합니다. HBO 드라마 실리콘밸리의 압축 알고리즘 자문으로도 알려져 있습니다.

[[Agent Memory - Characterization and System Implications of Stateful Long-Horizon Workloads]] 논문에 공저자로 참여했습니다. 상호작용 기록을 압축해 기억으로 저장하고 다시 꺼내 쓰는 문제는 본질적으로 정보의 표현과 압축이라는 그의 전공과 맞닿아 있습니다. 같은 논문에 [[Yasmine Omri]], [[Thierry Tambe]], [[Alex Pentland]], [[Marian Verhelst]]가 함께했습니다.
