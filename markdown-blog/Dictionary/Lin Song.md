Lin Song은 [[JD Explore Academy]] 소속 비전 연구자입니다. CVPR 2024에 발표된 *YOLO-World: Real-Time Open-Vocabulary Object Detection*(AILab-CVC/YOLO-World)의 공동·교신 저자로 가장 잘 알려져 있습니다. YOLO 계열에 vision-language modeling을 결합해 open-vocabulary detection을 실시간 수준에서 가능하게 만든 작업입니다.

이전에는 Tencent ARC Lab에서 컴퓨터 비전 연구를 이끌었고, 객체 검출과 open-world recognition을 연결하는 후속 연구(예: YOLO-UniOW 등)에 인용·언급됩니다.

[[OmniNFT - Modality-wise Omni Diffusion Reinforcement for Joint Audio-Video Generation|OmniNFT]]에서는 *project leader*로 표기됐습니다. 1저자 [[Guohui Zhang]]의 RL fine-tuning 연구 라인과 JD Explore Academy의 비디오 생성 자산을 연결해 프로젝트를 가동시킨 역할로 보입니다. USTC 1저자 + JD Explore Academy 시니어 + [[Nan Duan]] 디렉터로 이어지는 본 논문 협업 구조의 중간 허브에 위치합니다.
