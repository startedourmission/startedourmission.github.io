알리바바 AMAP(고덕지도) AMAP-ML/CV 라인의 video generation project lead로 활동합니다. AMAP-ML GitHub org에 공개된 video generation·평가·train-free 강화 작업 대부분이 [[우자홍]]의 이름을 끼고 있습니다.

대표 작업은 다음과 같습니다.

- MACE-Dance (SIGGRAPH 2026): motion·appearance를 cascaded expert로 분리하는 music-driven dance video 생성
- S²-Guidance (ICLR 2026): train-free diffusion 강화
- VMBench (ICCV 2025): perception-aligned video motion 벤치마크
- Omni-Effects: 통합·공간 제어 가능 visual effects 생성
- ImagerySearch (AAAI 2026): semantic dependency 제약을 넘는 test-time search

본 논문 [[Enhancing Train-Free Infinite-Frame Generation for Consistent Long Videos|MIGA]]에서는 project lead로 올라, AMAP video generation 라인의 *train-free* 톤을 그대로 이은 작업의 책임자 역할을 합니다. 1저자 [[펑샤오쿤]]이 CASIA 박사 인턴으로 합류해 핵심 기여를 했고, [[추샹샹]] 시니어 디렉터와 [[황카이치]] 학계 co-corresponding이 그 위를 받친 구조입니다.
