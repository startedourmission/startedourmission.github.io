## 개요

윤하이 퉁(Yunhai Tong)은 [[베이징대학교]]의 교수로 MSALab(Multimedia Semantic Analytics Lab)을 이끕니다. 2002년 베이징대학교에서 컴퓨터과학 박사 학위를 받았고, 미디어 지능 컴퓨팅·딥러닝·멀티모달 학습을 주 연구 분야로 합니다. 최근에는 멀티모달 확산 기반 모델 연구를 활발히 수행하며, 텍스트 추론·멀티모달 이해·텍스트-이미지 생성을 하나의 확산 아키텍처로 묶는 방향을 추구합니다.

[[PerceptionDLM - Parallel Region Perception with Multimodal Diffusion Language Models]]의 교신저자로, 확산 언어 모델을 멀티모달 인식 태스크로 확장하는 작업을 지도했습니다.
