[[Andrej Karpathy]]가 공개한 오픈소스 프로젝트로, 에이전트가 train_gpt.py 같은 학습 코드를 직접 수정하고 실행 결과를 분석하며 다음 실험을 자동으로 선택하는 파이프라인을 구현한다.

## 목표

자가 수정 루프에서 에이전트의 탐색 능력과 의사결정 수준을 측정하고 개선한다.

[[Parameter Golf]]와 유사한 목표를 지니며, 자동화된 [[하이퍼파라미터]] 최적화와 구조적 탐색의 경계를 넘어선 자율적 ML 엔지니어링을 추구한다.
