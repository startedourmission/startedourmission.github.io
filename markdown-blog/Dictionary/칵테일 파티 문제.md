# 칵테일 파티 문제

칵테일 파티 문제(Cocktail Party Problem)는 시끄러운 파티에서 여러 사람이 동시에 말할 때 마이크들이 혼합 신호를 녹음하는 상황처럼, 여러 독립 신호의 혼합으로부터 원본 신호들을 분리하는 문제이다.

## 핵심

- 독립 성분 분석(ICA)이 이 문제를 해결하기 위한 대표적인 알고리즘이다
- $m$개의 마이크가 $m$개의 혼합 신호를 수집하고, $m$개의 원본 신호를 복원하는 것이 목표이다
- 원본 신호들이 통계적으로 독립이고 비가우시안이라는 가정을 사용한다
- 음성 분리, 음악 분리, EEG 신호 분석 등에 실제 응용된다
- 혼합 행렬 $A$를 알 수 없다는 점에서 어려운 맹목 신호 분리(blind source separation) 문제이다

## 수식

관측: $x = As$, 복원 목표: $s = Wx = A^{-1}x$
