[[Claude]] 모델 라인업의 중간급(Sonnet) 모델로, 많은 엔드유저 애플리케이션에서 기본으로 사용되는 모델. 비용 효율과 응답 속도에서 우수하지만, 복잡한 자가 수정 루프나 장기 메모리 활용에는 제약이 있다.

## 메모리 활용 성능

[[Continual Learning Bench 1.0]]에서:
- **검증 커버리지**: 1단계 수준 (실패 기록만 가능)
- **메모리 참고**: 거의 하지 않음
- **개선 방안**: 명확한 메모리 사용 지시 필요

[[Claude Fable 5|Fable 5]]와 비교해 자동으로 메모리 프로토콜을 수립하지 못하며, 사용자가 명시적으로 가이드해야 한다.
