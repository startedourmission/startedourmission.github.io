v0는 Next.js 개발사 [[Vercel]]이 2023년 10월 공개한 AI 기반 UI 생성 도구다. 자연어 프롬프트만으로 React + Tailwind + shadcn/ui 컴포넌트와 페이지를 생성하고, 그 결과물을 Vercel 인프라에 즉시 배포할 수 있다.

초기에는 단일 컴포넌트 생성에 가까웠지만, 이후 멀티 파일 풀스택 앱 빌드, 데이터베이스 연결, 인증 통합 같은 기능이 추가되며 [[Lovable]]·[[Bolt.new]]와 함께 "프롬프트로 앱 만들기" 시장의 주요 플레이어로 자리 잡았다.

Vercel이 보유한 Next.js·AI SDK·AI Gateway 등과 자연스럽게 연결되는 점이 특히 프런트엔드 개발자에게 진입 장벽을 크게 낮춘다.
