[[딜렉 하카니튀르]]는 미국 일리노이대학교 어배너섐페인(UIUC) 컴퓨터과학과 교수입니다. 대화형 AI, 자연어·음성 처리, 음성 대화 시스템, 언어 처리용 머신러닝을 연구합니다. 2018년부터 2023년까지 아마존 알렉사 AI의 수석 주임과학자(senior principal scientist)로 기계와의 자연스러운 대화를 연구했습니다.

지식 기반 LLM을 오픈 도메인 대화 시스템에 적용해 알렉사 프라이즈 소셜봇에 실제로 투입했고, 알렉사 프라이즈 Simbot 챌린지를 설계했습니다. 300편이 넘는 논문과 100건이 넘는 특허를 보유합니다.

본 논문 [[ReasoningFlow - Discourse Structures for Understanding LLM Reasoning Traces]]에 시니어 저자로 참여했습니다. 추론 트레이스의 모니터링 가능성(monitorability)과 충실성(faithfulness)을 높인다는 이 논문의 응용 지향이, 대화 시스템을 실제 사용자에게 안전하게 배포해온 그의 경험과 맞닿아 있습니다. 같은 UIUC의 [[이진우]], [[줄리아 호켄마이어]]와 함께했습니다.
