Xiaomi LLM-Core 팀에서 MiMo 시리즈 전반의 사전학습·정렬 작업에 핵심 기여자로 이름을 올리는 시니어 연구자입니다. MiMo-7B 추론 모델, MiMo-VL 비전·언어 모델, MiMo-V2-Flash까지 Xiaomi가 공개한 주요 기술 보고서 대부분에 Core Contributor로 등재돼 있습니다.

산업계 LLM 팀의 시니어 위치에서 외부 학계와의 공동 작업을 매개하는 역할이 두드러집니다. 이번 [[Video2GUI - Synthesizing Large-Scale Interaction Trajectories for Generalized GUI Agent Pretraining|Video2GUI]]에서는 [[리수젠]](베이징대)와 함께 corresponding author로 올라, Xiaomi 측 자원과 모델 베이스(Qwen2.5-VL, MiMo-VL)를 데이터 파이프라인 쪽에 연결하는 역할을 맡았습니다. 1저자 [[슝웨이민]]이 인턴 기간 동안 본 작업을 수행했고, Xiaomi의 MiMo-VL이 그대로 베이스 모델 중 하나로 쓰였다는 점이 이 협업 구조를 잘 보여줍니다.
