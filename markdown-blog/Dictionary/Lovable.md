Lovable은 스웨덴 스톡홀름 기반 스타트업이 2024년 출시한 AI 풀스택 앱 빌더다. 같은 팀이 만든 오픈소스 프로젝트 GPT Engineer의 사업화 버전으로, 채팅 형태의 인터페이스에서 자연어로 요구사항을 말하면 React + [[Supabase]] 기반 풀스택 앱을 자동으로 빌드한다.

Figma 디자인 import, Stripe 결제 연동, 자체 도메인 배포 등 비개발자가 SaaS를 만들기 위해 필요한 부속 기능을 기본 제공한다. 출시 8개월 만에 ARR 1억 달러를 돌파하며 유럽에서 가장 빠르게 성장한 AI 제품으로 꼽힌다.

[[Bolt.new]]·[[v0]]와 함께 "프롬프트만으로 SaaS 만들기" 카테고리를 정의한 도구이며, 비기술 창업자·인디 메이커 사이에서 특히 점유율이 높다.
