Bolt.new는 브라우저 기반 IDE 회사 StackBlitz가 2024년 10월 공개한 AI 풀스택 앱 빌더다. 자체 기술 WebContainer 위에서 Node.js와 패키지 매니저가 그대로 동작하기 때문에, 모델이 생성한 코드를 별도 환경 없이 브라우저 안에서 즉시 실행·디버그한다.

자연어 프롬프트로 React·Vue·Svelte 등 주요 프레임워크 기반 풀스택 앱을 만들 수 있고, 결과물은 GitHub 푸시·Netlify·[[Vercel]] 배포로 바로 연결된다. 출시 후 첫 달 만에 ARR 400만 달러를 돌파하며 바이브 코딩 도구 가운데 가장 빠른 초기 성장을 기록한 제품 중 하나다.

브라우저에서 모든 게 끝나는 환경 덕분에 모바일·아이패드에서도 풀스택 개발이 가능하다는 점이 [[Cursor]]·[[Windsurf]] 같은 데스크톱 IDE와 다른 차별 포인트다.
