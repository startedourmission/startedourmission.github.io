GPT(Generative Pre-trained Transformer)는 [[OpenAI]]가 개발한 [[트랜스포머]] 디코더 기반 언어 모델 시리즈입니다. 2018년 GPT-1을 시작으로 GPT-2, GPT-3, GPT-4를 거치며 규모와 성능을 키워 왔습니다.

대규모 텍스트로 사전학습한 뒤 미세조정·정렬하는 패러다임을 대중화했고, [[ChatGPT]]를 통해 생성형 AI를 일상으로 끌어들였습니다.
