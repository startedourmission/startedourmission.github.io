Sakana Fugu는 [[Sakana AI]]가 2026년 6월 22일 출시한 멀티에이전트 오케스트레이션 시스템입니다. GPT나 Claude 같은 단일 거대 모델이 아니라, 여러 프런티어 모델을 문제마다 골라 조합하고 지휘하는 구조입니다. 실제 추론·생성은 뒤에 있는 외부 모델 풀이 맡고, Fugu 자체는 그들을 지휘하는 작은 LLM 하나와 오케스트레이션 레이어입니다.

Sakana는 이를 "하나의 모델 API로 제공되는 멀티에이전트 시스템"이라 부릅니다. 사용자에게는 OpenAI 호환 엔드포인트 하나로 보이지만, 내부에서는 여러 모델이 협업합니다. 지휘자가 강화학습으로 조합 전략을 직접 학습한다는 점, 자기 자신을 재귀적으로 호출해 지휘 전략을 다듬는다는 점이 특징입니다.

[[리온 존스]]가 "트랜스포머에 지쳤다"며 단일 아키텍처 의존을 비판해 온 [[Sakana AI]]의 노선과 맞닿아 있습니다. 더 큰 모델 대신, 이미 존재하는 최고 모델들을 잘 엮는 쪽에 베팅한 제품입니다. ICLR 2026 논문 Trinity와 Conductor가 기술적 기반입니다.

성능과 지연의 균형을 맞춘 Fugu, 정확도를 최대화한 Fugu Ultra 두 가지로 제공됩니다.
