**Spider2-V**는 Spider/Spider2 시리즈의 비전 버전으로, 데이터베이스 질문에 이미지(테이블, 차트, 스크린샷)가 포함된 멀티모달 SQL 생성 벤치마크입니다.

Spider2-V의 특징:
- **비전-SQL 결합**: 텍스트 질문 + 시각 증거(테이블 이미지, 차트) → SQL 생성
- **실제 시나리오**: 스프레드시트·데이터베이스 UI 스크린샷 포함
- **도메인 다양성**: 금융·의료·과학 데이터베이스
- **평가 지표**: 정확한 쿼리 매칭 및 실행 결과 일치성

[[MinerU2.5]]는 테이블 인식·추출에서 Spider2-V 같은 SQL 기반 벤치마크와도 호환되어야 하며, 테이블 구조 인식([[TEDS]] 메트릭)이 SQL 생성 정확도에 직결됩니다. 특히 크롭·회전된 표를 정확히 파싱하는 능력이 Spider2-V 같은 비전-SQL 작업의 전제 조건입니다.
