Amazon.com, Inc.는 미국의 다국적 전자상거래, 클라우드 컴퓨팅, 디지털 스트리밍 및 인공지능 기업입니다.

## 주요 사업

**전자상거래 (Amazon.com)**
- 온라인 소매 플랫폼 (의류, 전자제품, 도서 등)
- Prime 구독 서비스: 배송, 스트리밍, 음악 등 통합

**클라우드 (AWS, Amazon Web Services)**
- 세계 최대 클라우드 인프라 제공자
- EC2, S3, RDS, SageMaker 등 서비스
- 연간 수십억 달러 영업이익 창출

**디지털 스트리밍 (Prime Video, Music)**
- 영상 스트리밍 플랫폼
- 오디오 뮤직 서비스

**신규 사업**
- Project Kuiper: 위성 인터넷 (Blue Origin 자매 사업)
- 우주 산업 진출: SpaceX와 경쟁 구도

## 역사

- 1994년 Jeff Bezos가 온라인 서점으로 창업
- 1997년 나스닥 상장
- 2006년 AWS 출범 (현재 핵심 수익원)
- 2021년 Jeff Bezos 회장 퇴임, Andy Jassy CEO 취임

## 2026년 우주 전략

Blue Origin을 통한 로켓 기술 및 우주 관광 추진. Project Kuiper로 SpaceX Starlink와 경쟁하는 위성 인터넷 사업 중.
