훅(hooks)은 에이전트의 작업 흐름 중 미리 정해진 시점에 사용자가 등록한 명령을 자동으로 실행하는 장치입니다. 도구 호출 직전과 직후, 세션 시작과 종료 같은 지점에 끼워 넣어, 모델의 판단에 맡기지 않고 규칙으로 강제할 수 있습니다.

## 쓰임새

- 코드 수정 후 포매터나 린터를 자동 실행
- 민감한 명령을 차단하거나 승인 절차를 끼워 넣는 가드레일
- 작업 완료 시 알림 전송이나 로그 기록

[[Claude Code]] 같은 코딩 에이전트에서 [[MCP]], [[서브에이전트]]와 함께 자동화 수준을 높이는 핵심 도구로 쓰입니다.
