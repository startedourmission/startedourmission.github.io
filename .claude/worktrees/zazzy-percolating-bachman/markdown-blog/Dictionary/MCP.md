Model Context Protocol. [[Anthropic]]이 2024년 11월 오픈소스로 공개한 AI 에이전트 통합 표준입니다.

## 왜 필요한가

[[LLM]] 기반 에이전트가 이메일, 파일, 데이터베이스, API 같은 외부 시스템과 연결하려면 각 도구마다 별도의 플러그인을 만들어야 했습니다. MCP는 이 연결을 USB 포트처럼 표준화합니다. 도구는 MCP 서버를 구현하고, 에이전트는 MCP 클라이언트로 연결합니다. 한 번 만든 서버는 [[Claude]], [[Cursor]], Claude Code 등 MCP를 지원하는 모든 에이전트에서 동작합니다.

## 구조

- **MCP 서버**: 도구·리소스·프롬프트를 노출하는 프로세스. 파일 시스템, 슬랙, 깃허브, 데이터베이스 등이 각자의 서버를 가짐
- **MCP 클라이언트**: 서버에 연결해 도구를 호출하는 에이전트 측 컴포넌트
- **전송 계층**: 로컬 프로세스(stdio), HTTP, WebSocket

## 채택 현황

2025년 이후 주요 에이전트 플랫폼(Claude, Cursor, Windsurf, Claude Code)과 수천 개의 서드파티 서버가 MCP를 채택했습니다. 2026년 기준 에이전트 워크플로우 구성의 사실상 표준입니다.
